    Class AxoMessenger

    Provides mechanism for delivering static messages to the supervising applications (SCADA/HMI).Each messenger has a single placeholder for the message; it stores the most important message from the last context cycle.If there are more messages in a single cycle for the same messenger then less important messages are discarded and replaced bythe message with the category of higher severity.Messenger can send the data to context logger. There is a maximum limit of 10 different messages per messenger in 2 consecutive context cycles.The messages that exceed this limit won't be logged when OnEventRisen logging is set.Take this behaviour into account when designing your system.
