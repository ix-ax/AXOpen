    Class AxoStringBuilder

    Allows to concat strings using fluent interface. It's similar to C# StringBuilder classUsage as follows :

    ErrorString := _stringBuilder.Clear().Append('Error number: ').Append(ErrorNumber).Append('. Message: ').Append(ErrorMessage).Append('.').AsString();
    Inspired by Gerhard Barteling blogpost at https://www.plccoder.com/fluent-code/
